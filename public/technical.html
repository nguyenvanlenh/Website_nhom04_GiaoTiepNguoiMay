Hướng dẫn sử dụng Firebase

1) Tìm hiểu về Firebase

1.1 Firebase là gì?

Firebase là một nền tảng để phát triển ứng dụng di động và trang web, bao gồm các API đơn giản và mạnh mẽ mà không cần backend hay server.

1.2 Các ưu điểm của Firebase

  • Tạo tài khoản và sử dụng dễ dàng
  • Tốc độ phát triển nhanh
  • Nhiều dịch vụ trong một nền tảng
  • Được cung cấp bởi Google
  • Tập trung vào phát triển giao diện người dùng
  • Firebase không có máy chủ
  • Học máy (Machine Learning)
  • Tạo lưu lượng truy cập
  • Theo dõi lỗi
  • Sao lưu

2) Các bước để tạo một trang web với firebase.

B1: Chuẩn bị

1. Một tài khoản google

2. Cài đặt nodejs tại đây: nodejs.org


3. Cài đặc Firebase CLI bằng cách mở command line gõ lệnh "npm install -g firebase-tools".


Sau khi cài xong để kiểm tra chạy lệnh "firebase --version". nếu như hiện phiên bản như bên dưới hoặc cao hơn là đã thành công.


B2: Tiến hành tạo một trang web

1. Vào trang web firebase.google.com

2. Tiến hành đăng nhập vào firebase bằng tài khoản google.


3. Sau khi đăng nhập chọn get start.


4. Tạo một dự án mới.


5. Đặt tên cho dự án.


6. Chọn Carry on.


7. Chọn Default Account for Firebase.


8. Chọn vào đây để tạo website cho dự án của bạn.


9. Đặt tên cho website của bạn và chọn đăng ký.


10. Vậy là đã đăng ký thành công một web app trên firebase để chứa


11. Tiếp theo bạn cần tạo một thư mục trên máy tính của bạn để chứa dự


12. Mở cmd trong thư mục vừa tạo.


13. Đăng nhập vào firebase bằng lệnh "firebase login".


14. Tiếp tục gõ lệnh "firebase init hosting" để kết nối CLI tới tài khoản Firebase online.


15. Chọn "Use an existing project" và chọn tên dự án bạn đã tạo ở trên.


16. Trả lời các câu hỏi và chờ hiện ra "Firebase initialization complete!".


17. Tiếp theo cần gõ lệnh "firebase deploy" để đưa đưa code lên hosting của firebase là xong.


18. Truy cập vào đường link trong mục "Project console" hoặc "Hosting URL" để vào website của bạn.


Xong rồi nhé tiến hành vào code trang web của bạn nào!
